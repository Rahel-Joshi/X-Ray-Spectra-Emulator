# X-Ray-Spectra-Emulator
SURF 2024 Project seeking to emulate x-ray spectra utilizing machine learning in replacement of linear interpolation of X-ray spectra tables computed from physics models like XILLVER. X-ray spectra are nonlinear respective to their gamma, iron absorbance, ionization, energy cutoff, and inclination, so ML may be able to capture this nonlinearity.
